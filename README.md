# Breast-Cancer
Practice implementing various ML models in real world Breast Cancer Data 
## STEPS INVOLVED 
To evaluate the models on the real world data the following steps are meet:
1. Define Problem statement
2. Pre processing of data to bring useful insights from the data
3. Spliting data into train and validation sets
4. Train your model
5. Evaluate the performance of the model on data
6. Improve your model 
7. Deploy the model and monitor real world

## IN THIS MODEL
PROBLEM STATEMENT: To analyse the people which are prone to get Diagnosis.

We have extract the data and then pre processed the data and also performed EDA on the data set to increase our understanding of data.
Later, We have fit various models based on the problem i.e Logistic Regression , Decision Tree and KNN model.
The evaluation of each model is mentioned in the file.
